技术支持

课程与内容

VibeCode 当前版本包含系统课程、Prompt 库、上线清单和工具库。前 3 节课程可免费试听，完整内容需要通过 App Store 应用内购买解锁。

常见问题

为什么有些课程被锁定？

前 3 节课程为免费试听内容，其余课程需要购买 VibeCode Pro 后解锁。

购买后没有立即解锁怎么办？

  • 确认设备网络正常。
  • 完全关闭并重新打开 App。
  • 在“权益中心”点击“恢复购买”。
  • 确认当前 Apple ID 与购买时使用的 Apple ID 一致。

更换设备后如何恢复购买？

在新设备上使用购买时的 Apple ID 登录 App Store，然后打开 VibeCode，在“权益中心”点击“恢复购买”。

如何申请退款？

应用内购买由 Apple App Store 处理。请前往 reportaproblem.apple.com，使用购买时的 Apple ID 登录并按 Apple 流程提交申请。

是否需要账号登录？

当前版本不需要注册或登录。课程内容和基础学习状态保存在设备本地，购买权益由 Apple App Store 提供验证。

排查建议

  • 确保已从 App Store 更新到最新版本。
  • 检查网络连接，尤其是购买、恢复购买或打开外部链接时。
  • 尝试重启 App 或重启设备。
  • 如果问题与购买有关，请记录出现问题的大致时间、设备型号、系统版本和 App 版本。

联系我们

需要更多帮助？

请发送邮件至 support@vibecode.app。为了更快定位问题，建议在邮件中附上：

  • 设备型号与 iOS 版本；
  • VibeCode App 版本；
  • 问题发生的页面和操作步骤；
  • 如与购买有关，请说明是否已尝试“恢复购买”。

你也可以查看隐私政策与用户协议。
